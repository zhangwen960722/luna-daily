## 一、方向摘要（定稿前请先确认，可能微调）

- 工作台：一个服务组合 = 一个分组；运价员只维护**珠三角基准矩阵**（仓点组 × 重量段）。
- 其他交货地公布价 = 基准价 + **头程加价**（揽收装柜/提货费差），系统自动派生。
- 新增：只选组合，各交货地自动覆盖，不再逐交货地新增。
- **底层数据不变**：仍每交货地一条运价行（供查询命中），只是"维护入口"从手动 N 份 → 编基准 + 派生。
